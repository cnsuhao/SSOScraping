id=\"wikitext\">\n<h2>Welcome!</h2>\n<div><span class=\"rfloat\"> <a class=\"wikilink\" href=\"https://www.geany.org/Documentation/Screenshots\"><img src=\"https://www.geany.org/images/geany_main_small.png\" alt=\"Geany's main screen\" title=\"Geany's main screen\"></a></span></div>\n<p>Geany is a text editor using the GTK+ toolkit with basic features of an integrated development environment. It was developed to provide a small and fast IDE, which has only a few dependencies from other packages. It supports many filetypes and has some nice features.\nFor more details see <a class=\"wikilink\" href=\"https://www.geany.org/Main/About\">About</a>.\n</p>\n<p class=\"vspace\">Latest version is: 1.30.1\n<br clear=\"all\">\n</p>\n<div class=\"vspace\"></div><h2>News:</h2>\n<div class=\"blog-entry1\">\n<div class=\"blog-title\"><p><strong>Geany 1.30.1 is out!</strong>\n</p>\n</div>\n<div class=\"blog-timestamp\">Posted on Mar. 19, 2017 at 06:04 PM</div>\n<div class=\"blog-permalink\"><a class=\"wikilink\" href=\"https://www.geany.org/Main/20170319\">Permanent link</a></div>\n\n<p>Shortly after 1.30, we have released 1.30.1 which fixes <a class=\"urllink\" href=\"https://github.com/geany/geany/issues/1422\" rel=\"nofollow\">calltip and auto-completion popup placement with multi-monitor setups</a>.  We are sorry for this inconvenience.\n</p>\n<p class=\"vspace\">We also took the opportunity to include new translation updates that didn't make it into 1.30 in time: ca, de, el, es, sk.\n</p>\n<p class=\"vspace\">As usual, all downloads can be found on <a class=\"wikilink\" href=\"https://www.geany.org/Download/Releases\">Releases</a>.\n</p></div>\n\n<div class=\"vspace\"></div><div class=\"blog-entry2\">\n<div class=\"blog-title\"><p><strong>Geany 1.30 is out!</strong>\n</p>\n</div>\n<div class=\"blog-timestamp\">Posted on Mar. 05, 2017 at 03:06 PM</div>\n<div class=\"blog-permalink\"><a class=\"wikilink\" href=\"https://www.geany.org/Main/20170305\">Permanent link</a></div>\n\n<p>We are happy to announce a new release of Geany!\n</p>\n<p class=\"vspace\">For a comprehensive list of changes please see <a class=\"wikilink\" href=\"https://www.geany.org/Documentation/ReleaseNotes\">Release Notes</a>. A very detailed and complete list of changes can be found in the <a class=\"wikilink\" href=\"https://www.geany.org/Documentation/ChangeLog\">ChangeLog</a>.\n</p>\n<p class=\"vspace\">Some highlights:\n</p>\n<div class=\"vspace\"></div><ul><li>Initial accessibility support in the editor.\n</li><li>Fix scrolling on Wayland.\n</li><li>Fix Ctrl+X and Ctrl+C in non-Latin keyboard layouts (Forkest).\n</li><li>Update Scintilla to version 3.7.3.\n</li><li>Add Arduino custom filetype (<span class=\"wikiword\">SukkoPera</span>).\n</li><li>Updated translations: de, es, fr, it, lt, pt.\n</li></ul><p class=\"vspace\">We want to thank all developers, translators and everyone who contributed to this release with patches, feedback, bug reports and so on. Thank you!\n</p>\n<p class=\"vspace\">As usual, all downloads can be found on <a class=\"wikilink\" href=\"https://www.geany.org/Download/Releases\">Releases</a>.\n</p></div>\n\n<div 03:56 PM</div>\n<div class=\"blog-permalink\"><a class=\"wikilink\" href=\"https://www.geany.org/Main/20170305\">Permanent link</a></div>\n\n<p>We are happy to announce a new version of the combined Geany-Plugins release -- Following the Geany 1.30 release, Geany-Plugins 1.30 is being released.\n</p>\n<p class=\"vspace\">Some highlights\n</p>\n<div class=\"vspace\"></div><ul><li><span class=\"wikiword\">GeanyPy</span>: Sync with upstream project. \n</li><li>Lineoperations: Add support for selections\n</li><li>Markdown: Allow exporting generated Markdown as HTML\n</li><li><span class=\"wikiword\">PairTagHighlighter</span>: Add keybinding for jumping to matching tag\n</li><li><span class=\"wikiword\">ProjectOrganizer</span> Remember expanded directories in the sidebar when reloading\n</li><li>Treebrowser: Add a keybinding to follow current file\n</li><li>Treebrowser: Allow to also trigger menu with Shift+F10\n</li><li>Treebrowser: Don't change directory on project
