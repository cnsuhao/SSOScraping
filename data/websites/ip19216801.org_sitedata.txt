This address could be entered into browser address bar to access router settings page. There you can change router default password, Wi-Fi settings, security settings of your network and many more. Also on the settings page you can monitor connection of your router to the internet, reset router and connection settings or change firewall settings</p>\n<h3>Connecting to your router</h3>\n<p>To access you router settings page you don’t need active internet connection. You just need to type an IP address which is used by your router for administration. In case of Netgear and D-Link routers this address is 192.168.0.1 (sometimes misspelled as 192.168.o.1). For other routers this address could be different (for example for Linksys it is 192.168.1.1).</p>\n<h3>How to find out your router IP address.</h3>\n<p>To check if you router is actually using 192.168.0.1 IP address you need to perform following steps.</p>\n<div style=\"float:left;margin:10px 10px 10px 10px\">\n\n \n<ins class=\"adsbygoogle\" style=\"display:inline-block;width:336px;height:280px\" data-ad-client=\"ca-pub-3673974273889108\" data-ad-slot=\"5617522675\"></ins>\n\n</div>\n<h3>Default login for D-Link Router</h3>\n<p>Unless your internet provider changed default username and password following login details should work for 192.168.0.1:</p>\n<p>- Username: admin</p>\n<p>- Password: blank</p>\n<h3>Default login for Netgear Router</h3>\n<p>Unless your internet provider changed default username and password following login details should work for 192.168.0.1:</p>\n<p>- Username: admin</p>\n<p>- Password: password or 1234</p>\n<h3>&nbsp;How to change your router password</h3>\n<p>It is important that you change your router password so noone is able to access and reconfigure your router.
